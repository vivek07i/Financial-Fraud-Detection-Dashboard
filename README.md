# Advanced Financial Fraud Detection System

##  Overview
Developed an end-to-end fraud detection system using SQL, Python, and Machine Learning.

##  Features
- Analyzed 500K+ transactions
- Used advanced SQL (CTE, Window Functions)
- Built ML model (Logistic Regression) for fraud prediction
- Created Power BI dashboard for insights

##  Tech Stack
- SQL (CTE, Window Functions)
- Python (Pandas, Scikit-learn)
- Power BI

##  Results
- Identified high-risk users using ranking
- Detected unusual transaction spikes
- Improved fraud detection accuracy
